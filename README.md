# Iris Flower Classification

## Objective
Classify Iris flowers into Iris-setosa, Iris-versicolor, and Iris-virginica using machine learning.

## Dataset
Iris Dataset from Kaggle.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Algorithm
Decision Tree Classifier

## Accuracy
93.33%

## How to Run

```bash
pip install -r requirements.txt
python iris_classification.py
```
